## Como Executar

1. Clone o repositório:
   ```
   git clone https://github.com/SnDann/portfolio.git
   cd portfolio
   ```

2. Abra o arquivo `index.html` em um navegador moderno (Chrome, Firefox, Edge, etc.):
   - Duplo-clique no arquivo ou use `start index.html` no terminal (Windows).

O site é estático e não requer servidor ou dependências adicionais.

## Estrutura de Arquivos

- `index.html`: Página principal com seções de hero, projetos, sobre e contato.
- `style.css`: Estilos CSS com variáveis para tema escuro e responsividade.
- `README.md`: Este arquivo.

## Publicação (GitHub Pages)

1. Faça push do repositório para o GitHub na branch `main`.
2. No GitHub, acesse `Settings` → `Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Em `Branch`, escolha `main` e em `Folder`, selecione `/root`.
5. Aguarde alguns minutos até o site publicar em `https://sndann.github.io/Portfolio/`.
6. Atualize `og:url` em `index.html` com a URL publicada e, opcionalmente, defina uma imagem real em `og:image`/`twitter:image` (1200×630).
